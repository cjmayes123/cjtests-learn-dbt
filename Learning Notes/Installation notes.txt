INSTALLATION AND WORKING NOTES

-command will install dbt within the terminal.
-pip3 install dbt

- initialise dbt folder through:
 dbt init cjtests-learn-dbt

- creates the shell. you can cd into the learn-dbt folder and list (ls) the folder items

- create a database to connect to.
- database documentation https://docs.getdbt.com/docs/available-adapters

-- find the configuration file
dbt debug --config-dir

-- this profile connects to snowflake account

-- you need to cd into the learn-dbt file to run the debug
dbt debug
